## Objective
Set up the CMake project structure for Tower Tycoon with all Phase 1 dependencies.

## Requirements

### Directory Structure
```
vse-platform/
├── CMakeLists.txt          (root)
├── cmake/                  (CMake modules)
│   └── FetchDependencies.cmake
├── src/
│   ├── core/               (Layer 0 - interfaces)
│   ├── domain/             (Layer 1 - RealEstateDomain)
│   ├── content/            (Layer 2 - content loading)
│   ├── renderer/           (Layer 3 - SDL2)
│   └── main.cpp            (entry point - minimal SDL2 window)
├── include/
│   ├── core/
│   ├── domain/
│   ├── content/
│   └── renderer/
├── assets/
│   └── config/
│       └── game_config.json (placeholder)
├── tests/
│   └── CMakeLists.txt
└── .github/
    └── workflows/
        └── build.yml       (macOS build check)
```

### Dependencies (via CMake FetchContent)
- **SDL2** (2.28+)
- **SDL2_image**
- **SDL2_ttf**
- **EnTT** (latest, header-only)
- **Dear ImGui** (docking branch) + SDL2 backend
- **nlohmann/json** (latest)
- **msgpack-c** (MessagePack for C++)
- **spdlog** (logging)
- **Catch2** (v3, for unit tests)

### main.cpp Requirements
- Initialize SDL2
- Create 1280x720 window titled "Tower Tycoon"
- Main loop: handle quit event, clear screen (dark background), present
- Clean shutdown
- This is just to verify the build works. No game logic.

### CMake Requirements
- C++17 standard required
- macOS target (Apple Silicon + Intel)
- `-Wall -Wextra -Wpedantic` warnings
- Build type: Debug by default
- `set(CMAKE_EXPORT_COMPILE_COMMANDS ON)`

### GitHub Actions (`.github/workflows/build.yml`)
- Trigger: push to main/dev
- Runner: macos-latest
- Steps: checkout → install SDL2 via brew → cmake configure → cmake build
- Just build check, no tests yet

## Constraints (from CLAUDE.md)
- C++17 only (no C++20 features)
- No exceptions (`-fno-exceptions` flag)
- No hardcoded values in code
- File naming: PascalCase (e.g., `SimClock.h`)
- Interface prefix: `I` (e.g., `IGridSystem.h`)

## Expected Output
- All files listed above created and buildable
- `cmake -B build && cmake --build build` succeeds
- SDL2 window appears and closes cleanly
- GitHub Actions workflow file ready

## Unit Test
- Not required for this task (infrastructure only)

## Report Back When
- Build succeeds on local machine
- All directories created per structure above
- main.cpp shows SDL2 window
